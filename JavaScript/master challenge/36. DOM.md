# get by Name

```js

element.style.backgroundColor = "lightgreen";

console.log(fruits);
```

# checked property check
```js
let fruits = document.getElementsByName("fruits");

  

fruits.forEach((fruit) => {

  if (fruit.checked) {

    console.log(fruit.value);

  }

});
```

**output : apple**
